CSS Seçiciler

CSS'yi HTML'nin Farklı Bölümlerine Uygulama

1. Öğe seçici, öğeleri HTML etiket adlarına göre hedefler.

  1. Sınıf seçiciler, sınıf özelliğinin değerine göre öğeleri hedefler.
  2. Kimlik seçiciler, kimliğin değerine göre öğeleri hedefler özellik.
  3. Özellik seçiciler, öğeleri niteliklerine ve değerlerine göre hedefler.
  4. Evrensel seçici tüm öğeleri hedefler.
